Create a complete production-ready AI Resume & Career Copilot web application using Next.js 16 (App Router), TypeScript, and Tailwind CSS.

IMPORTANT: This project MUST be fully compatible with Vercel deployment and Replit AI generation.

====================
CRITICAL RULES (VERY IMPORTANT)
===============================

* DO NOT use Express

* DO NOT use any custom Node.js server

* DO NOT use pnpm workspace or monorepo

* DO NOT use drizzle, pino, or backend frameworks

* DO NOT create separate backend folders

* ONLY use Next.js App Router

* ONLY use /app/api routes for backend logic

* The project must be a single Next.js app

====================
CORE FEATURES
=============

1. AI Resume Generator

* Input fields: Name, Skills, Experience, Education
* On submit → call /api/generate-resume
* Use OpenAI API to generate a professional ATS-friendly resume
* Display result in a clean UI card

2. Resume Improver

* Textarea input
* Call /api/improve-resume
* Return improved professional version

3. Interview Prep

* Input: Role or skills
* Call /api/interview
* Return questions + answers
